# PPM Image Generator - Prompt Engineering Log

## Attempt 1
**Prompt**:  
Generate a C++ program that creates a simple PPM image with concentric rings in the center using vibrant colors.  
- Image must be at least 100×100 pixels  
- Must use ofstream to write to a .ppm file  
- Must use nested loops to generate pixel data  
- Must compile and run without errors  
- Output image must be viewable  
- Ensure the output file is saved in the program's relative directory  

**Result**:  
The program compiled successfully without errors. It generated a `.ppm` file in the correct directory. The image displayed correctly in a viewer and showed smooth, colorful concentric rings centered in the image.

**What I learned**:  
Being specific about constraints (file format, loops, file output location) helps ensure the generated code meets all requirements. Including details about visual output (concentric rings, vibrant colors) improves the quality of the result. Structured bullet points make the prompt clearer and easier for the model to follow.
